# Markview 동작 확인 샘플

이 문서는 Markview 확장이 제대로 동작하는지 시각적으로 검증하기 위한 샘플이다.
글을 읽는 느낌이 KEI editorial 풍인지 — 크림 배경, 내비 본문, 조용한 타이포 —
직접 눈으로 확인하는 것을 목적으로 한다.

## 1. 본문과 강조

본문 17px, 줄 간격 1.85. **굵은 강조**와 *이탤릭*, 그리고 [링크 스타일](https://example.com)이
어떻게 보이는지 확인하자. 인라인 코드는 `transformer`, `ensemble` 처럼 표기된다.

## 2. 수식 렌더링 (KaTeX)

인라인: 로렌츠 방정식의 한 식은 $\dot x = \sigma(y - x)$ 이다.

디스플레이:

$$
\begin{aligned}
\dot x &= \sigma(y - x) \\
\dot y &= x(\rho - z) - y \\
\dot z &= xy - \beta z
\end{aligned}
$$

## 5. 코드 블록

```python
def lorenz(state, sigma=10, rho=28, beta=8/3):
    x, y, z = state
    return [sigma * (y - x),
            x * (rho - z) - y,
            x * y - beta * z]
```

## 7. 각주

[^1]: 첫 번째 각주. 여기에 상세한 설명이나 참고문헌을 적는다.
